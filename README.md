# SimpleShop Application

### Выполненено:

* Регистрация и авторизация по JWT токену
* Функционал админа расширяет функционал пользователя
* Пользователь может купить товар, если на его балансе достаточно средств. Админ может пополнить баланс любого пользователя
* Пользователь может ставить отзывы и оценки только товару, который он купил
* Пользователь может подать заявку на создание организации
* После рассмотрения заявки админом организация получает статус активной и закрепляется за пользователем, который подал заявку
* Пользователь может добавлять товары в свою организацию
* Пользователь может читать уведомления, отправленные ему админом. Админ может отправить уведомление любому пользователю
* Пользователь может получить текущую информацию о себе
* Админ может менять статус пользователя
* Реализована система скидок с проверкой на валидность
* Пользователю, чтобы совершить покупку необходимо добавить товар в корзину. История покупок сохраняется
* Пользователь может вывести средства со своей организации на свой баланс
* В качестве UI используется Swagger
* Порт по умолчанию 8080
* http://localhost:8080/swagger-ui/index.html#/ - URL для браузера
* После запуска приложения инициализируется пользователь с ролью админа, логином - 'admin', паролем - '123QWEasd'
